# PyTorch Deep Learning Practice

# 概要 (Overview)
書籍「最短コースでわかる PyTorch &深層学習プログラミング」を参考に、自分なりに深層学習の実装をしています。

# 実装・実験リスト (Projects)
- 活性化関数の役割の検証(activation-functions/)
- Irisデータセットによる2値分類(classification/iris_binary_classification)
- MNISTデータセットによる数字認識(mnist/train_mlp.py)

# 環境
- OS:macOS(Apple M3)
- Python:3.12.2
- 必要なライブラリは pip install -r requirements.txt でインストール可能です。
